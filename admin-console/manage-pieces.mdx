## Show Specific Addons in Project

If you go to **Addons Settings** in your project, you can manage which addons you would like to be available to your users. 

![Manage Addons](/resources/screenshots/manage-addons.png)
![Manage Addons](/resources/screenshots/manage-addons-2.png)
## Install Addon

- Go to **Platform Admin -> Setup -> Addons** and hit Install addons.
- You can choose to install a addon from NPM or upload a tar file directly for private addons.
- You can check the [Sharing Addons Doc](/developers/sharing-addons/overview) for more info.

![Manage Projects](/resources/screenshots/install-addon.png)
